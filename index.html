Cheap communication tools

Slack has awesome integrations, especially with source control and task management apps, that it's become an integrated part of one of my projects.

Suggested alternatives:

  • Hipchat is another one of those that is an awesome alternative to slack.
  • setup an Irc or a forum instead if you're up for it.

Monitoring apps

  • Pingdom
  • New relic

Bug tracking

You can use in built tools in bitbucket or github to track issues.

Slack commands to deploy your app

One way to get things done from slack is something called a slack slash commands. You just need a server that accepts a POST from slack and does something when it receives that.
